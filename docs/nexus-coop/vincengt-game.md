# 勇敢者游戏

战争的闹剧结束后，科普卢星区的种种异相表明一切还未结束。参与众多重要事件的战役任务，成为扭转末日浩劫的关键吧。

该模式下，突变因子选择将会禁用，勇敢者游戏将在指定地图上启用相应的三项突变因子。请小心，该模式对设备的性能要求较高，且游戏难度极高。
